# Bhrugu React Redux

##### React.js website

react:"^17.0.2"

### Packages used

- Ant design
- React Bootstrap
- Redux
- React Router
- Styled component

### Build

requires [Node.js](https://nodejs.org/) v16.3.0 to run.

to install packages

```sh
yarn
```

to run the project...

```sh
yarn start
```

the project will run on port 3000
